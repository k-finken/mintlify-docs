# Snippet Import Test Suite

This test suite verifies that the updated `mint dev` CLI correctly handles all snippet import patterns.

## Test Checklist

| Test Case                                    | File                                                                                 | Status |
| -------------------------------------------- | ------------------------------------------------------------------------------------ | ------ |
| Existing snippet imports                     | All test files                                                                       | ⏳     |
| Absolute path imports (`/snippets/...`)      | [test-absolute-imports](/test-snippets/test-absolute-imports)                        | ⏳     |
| Relative path imports (`../snippets/...`)    | [test-relative-imports](/test-snippets/test-relative-imports)                        | ⏳     |
| Nested absolute path imports                 | [test-absolute-imports](/test-snippets/test-absolute-imports)                        | ⏳     |
| Nested relative path imports                 | [test-relative-imports](/test-snippets/test-relative-imports)                        | ⏳     |
| Deep relative imports (`../../snippets/...`) | [test-deep-relative-imports](/test-snippets/nested-tests/test-deep-relative-imports) | ⏳     |
| Mixed imports (absolute + relative)          | [test-mixed-imports](/test-snippets/test-mixed-imports)                              | ⏳     |
| Variable exports                             | All test files                                                                       | ⏳     |

## How to Run Tests

1. Start the dev server:

```bash
mint dev
```

2. Navigate to each test page and verify:

   - All snippet content renders correctly
   - No import errors in the console
   - Props are passed correctly
   - Variable exports work

3. Update the checklist above with ✅ for passing tests or ❌ for failures.
